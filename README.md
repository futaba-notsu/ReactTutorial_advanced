# 概要
[ReactTutorial](https://ja.reactjs.org/tutorial/tutorial.html) を React＋Redux (ts) で実装する

# 内容
ReactTutorialに以下を追加(修正)
- ディレクトリ構成修正
- TypeScript で実装
- Redux store で変数管理
- css : StyledComponent 使用

# 起動
1. `cd my-app`  
my-app へ移動

2. `yarn`  
yarn.lock の読み込み

3. `yran start`  
[http://localhost:3000](http://localhost:3000) がブラウザで開く
